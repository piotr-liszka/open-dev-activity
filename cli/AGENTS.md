# CLI Architecture with Commander.js and TypeScript

A comprehensive guide for building scalable, agent-based CLI programs using Commander.js, Node.js, and TypeScript.

# Typescript

```
src/
├── commands/
│   ├── ...
├── github.ts

├── types.ts
├── auth.ts
├── working-time.ts
```

## Code style

- Use 2 spaces for indentation
- Use single quotes for strings
- Use trailing commas
- Use const for variables that don't change
- Use let for variables that change
- Use const for function parameters
